Welcome to Elsys Prosjekt Gruppe 14's documentation!
====================================================

Our project will simplify the task of administrating the medical equipment inventory at St. Olavs Hospital. The task was to create a
system that flags when inventory runs low and to generate purchase orders to ensure availability.

We have worked with `St. Olavs, Akutten <https://www.stolav.no/avdelinger/klinikk-for-akutt-og-mottaksmedisin/mottaksavdeling/>`_ on the task and want to thank for all supervision and good discussions.  Etc. ... 

.. image:: ./images/arduino.jpeg
  :width: 400
  :alt: Arduino board


Create the Conda environment needed and run the code with:

.. code-block:: bash

   $ conda env create -f elsys.yml
   $ conda activate elsys
   $ python Tkinter.py
